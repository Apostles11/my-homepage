$$\nabla_\mu T^{\mu\nu}=0,\qquad R_{\mu\nu}-\tfrac12 g_{\mu\nu}R=\tfrac{8\pi G}{c^4}T_{\mu\nu}.$$

2. 代码高亮（highlight.js）

例如一段 JavaScript：


// 计算洛伦兹因子 gamma
function gamma(v, c=299792458) {
  if (v >= c) throw new Error("v must be < c");
  return 1 / Math.sqrt(1 - (v*v)/(c*c));
}
console.log(gamma(1000));
